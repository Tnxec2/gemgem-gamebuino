# GemGem Gamebuino Classic Game
![](assets/gemgem.gif)

## Description
A Bejeweled clon game for Gamebuino Classic platform.

## Controls
### Game Screen
Arrows: move cursor
A: Mark Gem to swipe / swipe marked
B: Pause / Hint
C: Exit with save game

### GameOver Screen
A: show / hide gameover dialog
B: start new game
C: Exit game
